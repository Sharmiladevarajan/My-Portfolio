## ğŸ“¦ Installation & Setup

### Prerequisites
- Node.js 18.0 or higher
- npm, yarn, pnpm, or bun package manager

### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sharmiladevarajan/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## ğŸ¨ Customization

### Updating Personal Information

1. **Hero Section**: Edit `src/components/Hero.tsx`
   - Update name, roles, and description
   - Modify social media links

2. **About Section**: Edit `src/components/About.tsx`
   - Update personal background
   - Modify contact information

3. **Experience**: Edit `src/components/Experience.tsx`
   - Add/update work experiences
   - Modify achievements and technologies

4. **Skills**: Edit `src/components/Skills.tsx`
   - Update skill categories and levels
   - Add new technologies

5. **Projects**: Edit `src/components/Projects.tsx`
   - Add new projects
   - Update project descriptions and technologies

### Styling

- **Colors**: Modify Tailwind CSS classes throughout components
- **Fonts**: Update fonts in `src/app/layout.tsx`
- **Layout**: Adjust responsive breakpoints and spacing
